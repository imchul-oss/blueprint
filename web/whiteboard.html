저장 · 백엔드

첨부 자료의 저장 위치를 선택합니다. 로컬 IndexedDB는 브라우저 내에서 큰 파일을 빠르게 보관합니다(다른 PC와 공유 안 됨). 로컬 파일시스템은 Chrome/Edge에서 디렉토리를 지정해 파일로 저장합니다. URL은 외부 호스팅(Supabase Storage, S3, R2 등)에 직접 업로드한 파일의 링크만 첨부합니다.
BP 자체 백엔드는 SaaS 운영 시 Supabase/Postgres/Turso 로 분리됩니다.

AI · LLM 연결

자연어 명령·자동 비평·LLM 응답 통합에 사용됩니다.
Provider 선택 + API Key 입력 후 좌측 AI 패널에서 자연어 명령(예: "추가 feature: 소셜 로그인, 우선순위 P0") 입력하면 propose 큐로 응답이 들어옵니다.
UBP 본질 보호: LLM 응답은 노드 body / propose 큐로만 통합되며, .pptx · PRD 등 산출물 자체 렌더는 하지 않습니다.
LLM 호출은 사용자가 명시 트리거한 1회만 — 자동 반복 운행 없음.
